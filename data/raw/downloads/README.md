# Official source downloads

Store publisher PDFs, supplementary data, or other **official** acquisition files here.

## Rules

- One checksum file per downloaded artifact: `<filename>.sha256` containing the hex digest.
- Record filename, `source_id`, download date, and checksum in `docs/extraction_log.md`.
- Do not commit paywalled content without license clearance; large binaries may stay local-only.
- Do not use unofficial mirrors (e.g. ResearchGate figure scrapes) as primary sources.
